AI Prompt Generator Pro - Portable Version
===========================================

Version: 2.0.0

Quick Start:
1. Double-click "Start_Application.bat" or "PromptGeneratorPro.exe"
2. The application will start immediately

No installation required!
- No Python installation needed
- No dependencies to install
- Works from any location (USB drive, network drive, etc.)

Features:
- Modern dark theme UI
- Multiple input types (text, PDF, CSV, JSON, URLs)
- Interactive prompt builder
- Advanced prompt engineering techniques
- Prompt library and management
- Export in multiple formats

System Requirements:
- Windows 10/11 (64-bit)
- 2 GB RAM minimum
- 100 MB free disk space

For more information:
- See QUICK_START.md for usage guide
- See LICENSE for license information
- Visit https://github.com/ai-prompt-generator/pro

Troubleshooting:
If the application doesn't start:
1. Check that you have Windows 10/11 64-bit
2. Try running as administrator
3. Check Windows Event Viewer for errors
